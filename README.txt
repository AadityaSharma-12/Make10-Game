
HOW TO RUN:
1. Ensure Python 3.8+ is installed
2. Install dependencies: 
   pip install pygame
3. Run the game: 
   python main.py

CONTROLS:
- Click number buttons (1-9) to answer
- Press + or - to switch between addition/subtraction
- Goal: Combine numbers to make 10
